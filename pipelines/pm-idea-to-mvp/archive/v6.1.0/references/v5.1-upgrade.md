# v5.1 Upgrade Procedure (2026-06-11)

## Directory Restructure

v5.1 moved from `workflow/pm-idea-to-mvp/` to `pipelines/pm-idea-to-mvp/` and introduced new top-level directories:

```
ttmens-skills/
├── pipelines/pm-idea-to-mvp/     # Main pipeline (was workflow/)
├── domains/                       # Core skills by domain
│   ├── agents/                    # subagent-driven-development, writing-plans
│   ├── design/                    # design-system-md, ui-acceptance-review
│   ├── engineering/               # requesting-code-review, test-driven-development
│   ├── product/                   # c4-architecture, docs-hygiene, grill-*, openspec, etc.
│   └── qa/                        # dogfood
├── borrowed/                      # (auto-populated from vendor/)
├── profiles/                      # hermes-opencode, hermes-plan, deep-research
├── scripts/                       # check_docs_ssot.py, ui_acceptance.py, install_skills.py
└── vendor/                        # Git submodules
    ├── phuryn-pm-skills/          # PM framework skills
    └── knowledge-work-plugins/    # Engineering/ops/data skills
```

## Upgrade Steps

### 1. Pull latest and init submodules

```bash
cd /d/workspace/ttmens-skills
git pull origin main
git submodule update --init --recursive
```

### 2. Sync pipeline

```bash
rm -rf "D:/hermes-data/skills/workflow/pm-idea-to-mvp"
mkdir -p "D:/hermes-data/skills/pipelines"
cp -r "D:/workspace/ttmens-skills/pipelines/pm-idea-to-mvp" "D:/hermes-data/skills/pipelines/"
```

### 3. Sync supporting directories

```bash
cp -r "D:/workspace/ttmens-skills/scripts" "D:/hermes-data/skills/"
```

Install script:

```bash
mkdir -p "D:/hermes-data/skills/borrowed"

# phuryn skills
declare -A SKILL_MAP=(
  ["pm-identify-assumptions-new"]="identify-assumptions-new"
  ["pm-opportunity-solution-tree"]="opportunity-solution-tree"
  # ... (full map above)
)

for pm_name in "${!SKILL_MAP[@]}"; do
  dir_name="${SKILL_MAP[$pm_name]}"
  src=$(find "D:/workspace/ttmens-skills/vendor/phuryn-pm-skills" -type d -name "$dir_name" | head -1)
  [ -n "$src" ] && cp -r "$src" "D:/hermes-data/skills/borrowed/$pm_name"
done

# knowledge-work skills
for skill in deploy-checklist code-review testing-strategy incident-response runbook system-design sql-queries; do
  src=$(find "D:/workspace/ttmens-skills/vendor/knowledge-work-plugins" -path "*/skills/$skill" -type d | head -1)
  [ -n "$src" ] && cp -r "$src" "D:/hermes-data/skills/borrowed/kw-$skill"
done
```
