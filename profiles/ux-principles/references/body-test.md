# UX Body Test (excerpt)

Adapted from lgboim/ux-body-test — quick human-factor checks for desktop web flows.

| Check | Pass criteria |
|-------|----------------|
| Reach | Primary actions within thumb/mouse comfort zone (no excessive corner hunting) |
| Scan | F-pattern or Z-pattern supported; hero CTA visible without scroll on 1280×800 |
| Load | No layout shift on first interaction (CLS-friendly skeletons) |
| Focus | Tab order follows visual order; focus ring visible |
| Recovery | Error states explain fix in one sentence + one action |

Use during **journey** dogfood walk; log failures as P1 in `UX-REVIEW.md`.
